<p>A material styled, very light and fast compass application for Android. Built with vector-graphics, so it should look the same sharp on everything from smart-watches till hd screens. Requires no internet connection, but a working compass sensor and have no ads. Compatible with all versions above Android 4.0.3 IceCreamSandwich.</p>
